# ✅ VerificAI

A static web app that turns any news story, headline, or URL into a structured fact-checking prompt, ready to paste into your favourite AI.

No build step, no backend, no sign-up. Everything runs client-side.

## Running locally

Just open `index.html` in a browser — no server required.

## How it works

1. Paste a news snippet, headline, or URL into the input field.
2. Click **Generate prompt** — a structured verification prompt is built in your browser.
3. Copy the prompt or click one of the platform buttons to open ChatGPT, Gemini, Perplexity, or Claude directly.

## Languages

UI and prompts are available in **Spanish** and **English**. The language is auto-detected from the browser and can be switched at any time.

## Stack

- Plain HTML + JavaScript
- [Tailwind CSS](https://tailwindcss.com) via Play CDN
- Google Fonts (Inter)
